#VueJS Arq Front Application

# Install
    > npm install

The API used in this example is a Node.js express API.

# Run develop mode
    > npm run dev

Navigation to http://localhost:8080 to view the site

# Push on Heroku
    > npm run build
    > git push heroku master
    > go to https://arq-front-vue.herokuapp.com
